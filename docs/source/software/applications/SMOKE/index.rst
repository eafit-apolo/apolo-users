SMOKE
=====

SMOKE is primarily an emissions processing system designed to create gridded,
speciated, hourly emissions for input into a variety of air quality models such as CMAQ,
REMSAD, CAMX and UAM. SMOKE supports area, biogenic, mobile (both onroad and nonroad),
and point source emissions processing for criteria, particulate, and toxic pollutants. [1]_

Curl
====

Curl [1]_ is used in command lines or scripts to transfer data. curl is also used in cars, television sets, routers,
printers, audio equipment, mobile phones, tablets, settop boxes, media playersand is the Internet transfer engine
for thousands of software applications in over ten billion installations.

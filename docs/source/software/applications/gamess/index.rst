
******
gamess
******

Description
----------------

GAMESS is a program to work with ab initio groups of molecular quantum chemistry. Briefly
GAMESS can compute Honda SCF functions within the range of RHF, ROHF, UHF, GVB, and MCSCF.

- http://www.msg.ameslab.gov/GAMESS/capabilities.html

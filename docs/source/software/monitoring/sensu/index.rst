Sensu
======

The Sensu monitoring event pipeline [1]_ empowers businesses to automate their monitoring workflows and gain deep visibility into their multi-cloud infrastructure, from Kubernetes to bare metal.

This Tool is used in the computing center to monitor Memory usage, Service availability, CPU load, and Temperature status.

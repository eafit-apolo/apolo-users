Ruby
====

[1]_ Ruby is an interpreted, high-level, general-purpose programming language. It was designed and developed
in the mid-1990s by Yukihiro "Matz" Matsumoto in Japan.

Ruby is dynamically typed and uses garbage collection. It supports multiple programming paradigms,
inclding procedural, object-oriented, and functional programming. According to the creator, Ruby was
influenced by Perl, Smalltalk, Eiffel, Ada, Basic, and Lisp.

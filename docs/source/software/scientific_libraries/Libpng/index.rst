.. _Libpng:

Libpng
======

The libpng [1]_ package contains libraries used by other programs for reading and writing PNG files.
The PNG format was designed as a replacement for GIF and, to a lesser extent, TIFF, with many improvements
and extensions and lack of patent problems.

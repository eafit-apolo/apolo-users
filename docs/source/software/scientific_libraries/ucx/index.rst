Unified Communication X(UCX)
=============================

Unified Communication X (UCX)
is an open-source, high-performance communication framework for data-centric, HPC, and AI applications. It optimizes data movement between CPUs, GPUs, and network hardware (InfiniBand, TCP). UCX accelerates messaging (MPI), Remote Memory Access (RMA),
and atomic operations, providing a unified API for high-speed networking.

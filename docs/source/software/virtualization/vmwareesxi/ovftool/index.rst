============
**OVF TOOL**
============

**VMware OVF Tool** is an utility that allows the system administrator to package or unpackage virtual machines using the Open Virtualization Format from the command line. [1]_

In this entry we will cover the steps needed to package a VM controlled by a VMware host, by connecting to the host from your PC.
